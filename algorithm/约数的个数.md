# 约数的个数

## 题目描述

输入n个整数,依次输出每个数的约数的个数

## 输入描述:

```
输入的第一行为N，即数组的个数(N<=1000)
接下来的1行包括N个整数，其中每个数的范围为(1<=Num<=1000000000)
```

## 输出描述:

```
可能有多组输入数据，对于每组输入数据，
输出N行，其中每一行对应上面的一个数的约数的个数。
```

## 解答

```
#include <iostream>
using namespace std;

int countApproximate(int n) {
	int count = 0, i;
	for (i = 1;i*i < n;i++) {  // i*i < n 减小时间复杂度
		if (n % i == 0)
			count+=2;  // 每次多两个约数
	}
    if(i*i == n) count++;  // 注意最后的i
	return count;
}

int main() {
	int N,n;
	while (cin >> N) {
		while (N-- > 0) {
			cin >> n;
			cout << countApproximate(n) << endl;
		}
	}
}
```
