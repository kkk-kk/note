# 分解质因数

## 题目描述

求正整数N(N>1)的质因数的个数。 相同的质因数需要重复计算。如120=2*2*2*3*5，共有5个质因数。

## 输入描述:

```
可能有多组测试数据，每组测试数据的输入是一个正整数N，(1<N<10^9)。
```

## 输出描述:

```
对于每组数据，输出N的质因数的个数。
```

## 解答

```c++
#include <iostream>
#include <cmath>
using namespace std;

int main(){
    long N,count = 0,i = 2,n;
    cin >> N;
    n = sqrt(N);  // 使用开平方减小循环次数
    while(i<n){ 
        if(N%i == 0){
            count++;
            //cout << i << ' ';  // 输出质因数
            N = N/i;
        }
        else i++;
    }
    count = N>1?count+1:count;  // **如果N>1，代表N也是质因数，且是最后一个**
    cout << count;
}
```
